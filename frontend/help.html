Админ-панель

Вход в админ-панель

  1. Перейдите на главную страницу сайта
  2. Нажмите кнопку "Админ-панель" в верхнем меню
  3. Введите логин и пароль администратора
  4. Нажмите "Войти"

Создание мероприятия

  1. В админ-панели нажмите кнопку "Создать событие"
  2. Заполните вкладку "Основные":
    • Название мероприятия - укажите название вашего события
    • Дата события - выберите дату проведения
    • Время начала - укажите время начала мероприятия
    • Описание - добавьте описание мероприятия (необязательно)
  3. Настройте вкладку "Приватность":
    • Требуется модерация фото - включите, если хотите проверять фото перед публикацией
    • Доступ к просмотру - выберите, кто может просматривать галерею (Публичный/По паролю)
    • Если выбран доступ "По паролю", укажите пароль для доступа к галерее
    • Включить уведомления в Telegram - включите для получения уведомлений о новых фото на модерации
    • Если включены Telegram уведомления, укажите Telegram username и порог количества фото
  4. Нажмите "Сохранить" для создания мероприятия

Важно: Вкладка "Брендинг" доступна только после создания мероприятия. Вы сможете настроить внешний вид страницы QR-кода позже.

После создания мероприятия

После создания мероприятия вы можете редактировать его настройки, нажав кнопку "Настройки" рядом с мероприятием в списке событий. В модальном окне настроек доступны три вкладки: "Основные", "Приватность" и "Брендинг".

Вкладка "Основные"

Во вкладке "Основные" вы можете изменить:

  • Название события - изменить название мероприятия
  • Дата события - изменить дату проведения
  • Время начала - изменить время начала мероприятия
  • Описание - изменить или добавить описание мероприятия

После внесения изменений нажмите "Сохранить".

Вкладка "Приватность"

Во вкладке "Приватность" вы можете настроить:

  • Требуется модерация фото - включить или отключить модерацию загружаемых фото и видео
  • Доступ к просмотру - изменить тип доступа к галерее:
    • Публичный - любой может просматривать галерею
    • По паролю - для просмотра галереи требуется ввести пароль
  • Пароль для доступа к галерее - установить или изменить пароль (отображается только при выборе "По паролю")
  • Включить уведомления в Telegram - включить или отключить уведомления о новых фото на модерации
  • Telegram username - указать username пользователя Telegram для отправки уведомлений (без символа @)
  • Порог количества фото для уведомления - указать минимальное количество фото на модерации, при котором будет отправлено уведомление

Примечание: Пароль для доступа к галерее действителен в течение 3 часов после ввода пользователем.

Вкладка "Брендинг"

Вкладка "Брендинг" доступна только после создания мероприятия. Здесь вы можете настроить внешний вид страницы QR-кода:

  1. Откройте настройки мероприятия (кнопка "Настройки" рядом с мероприятием)
  2. Перейдите во вкладку "Брендинг"
  3. Настройте внешний вид:
    • Цветовая схема - выберите цвет фона для страницы QR-кода
    • Логотип - загрузите логотип для отображения на странице QR-кода (необязательно)
      • Поддерживаются форматы: JPG, PNG, WebP
      • Максимальный размер файла: 10 МБ
      • Максимальная ширина отображения: 200px
    • Фоновое изображение - загрузите фоновое изображение для страницы QR-кода (необязательно)
      • Поддерживаются форматы: JPG, PNG, WebP
      • Максимальный размер файла: 10 МБ
  4. Вы можете удалить логотип или фоновое изображение отдельно, не затрагивая другие элементы брендинга
  5. Нажмите "Сохранить" для применения изменений

Важно: Цвет фона и фоновое изображение применяются только к странице QR-кода. Галерея всегда имеет черный фон, независимо от настроек брендинга.

Модерация фотографий

  1. Перейдите во вкладку "Модерация" в админ-панели
  2. Вы увидите список всех фотографий и видео, ожидающих модерации
  3. Для просмотра фото/видео нажмите кнопку "Просмотр"
  4. Выберите фотографии/видео, которые хотите одобрить или отклонить:
    • Используйте чекбоксы для выбора отдельных элементов
    • Используйте кнопку "Выбрать все" для выбора всех элементов
  5. Нажмите "Одобрить выбранные" или "Отклонить выбранные"

В модерации отображаются только те фото и видео, для которых включена модерация в настройках мероприятия.

Аналитика

  1. Перейдите во вкладку "Аналитика" в админ-панели
  2. Просмотрите статистику:
    • Общее количество фото - количество всех загруженных фотографий
    • Общее количество видео - количество всех загруженных видео
    • Общий размер фото - суммарный размер всех фотографий
    • Общий размер видео - суммарный размер всех видео
    • Общий суммарный размер - общий размер всех медиафайлов в ГБ
  3. Изучите график загрузок - он показывает динамику загрузки фото и видео по времени

Управление мероприятиями

  1. Во вкладке "События" вы увидите список всех мероприятий
  2. Для каждого мероприятия доступны действия:
    • Настройки - редактирование параметров мероприятия
    • QR-код - просмотр и скачивание QR-кода
    • Галерея - переход к галерее мероприятия
    • Удалить - удаление мероприятия

Галерея

Доступ к галерее

  1. Отсканируйте QR-код мероприятия с помощью камеры телефона
  2. Или перейдите по прямой ссылке на галерею (если она была предоставлена)
  3. Если галерея защищена паролем, введите пароль в появившемся окне
  4. Пароль действителен в течение 3 часов после ввода

Просмотр фотографий

  1. В галерее перейдите во вкладку "Фото"
  2. Просмотрите все фотографии в виде сетки
  3. Нажмите на фотографию для просмотра в полном размере
  4. В режиме просмотра:
    • Используйте стрелки влево/вправо для навигации между фото
    • Нажмите на крестик или вне области фото для закрытия
    • Используйте кнопку "Поделиться" для отправки фото через мессенджеры
    • Нажмите на сердечко, чтобы поставить лайк

Просмотр видео

  1. В галерее перейдите во вкладку "Видео"
  2. Просмотрите все видео в виде сетки
  3. Нажмите на видео для воспроизведения
  4. Видео воспроизводится прямо на странице (не в полноэкранном режиме)
  5. Используйте кнопку "Поделиться" для отправки видео через мессенджеры

Загрузка фотографий и видео

  1. В галерее нажмите кнопку "Загрузить фото" или "Загрузить видео"
  2. Выберите файлы с вашего устройства:
    • Поддерживаемые форматы фото: JPG, JPEG, PNG, WebP
    • Поддерживаемые форматы видео: MP4, MOV, AVI
  3. Если включена модерация, ваши фото/видео появятся в галерее после одобрения администратором
  4. Если модерация отключена, фото/видео появятся в галерее сразу после загрузки

Live Фотостена

  1. Если в галерее более 5 фотографий, появится кнопка "Live Фотостена"
  2. Нажмите на кнопку для перехода к Live Фотостене
  3. Фотостена автоматически отображает фотографии в случайном порядке
  4. Каждое фото отображается в течение 4 секунд с плавными переходами
  5. Новые фотографии появляются автоматически без перезагрузки страницы

Страница QR-кода

Доступ к странице QR-кода

  1. В админ-панели откройте настройки мероприятия
  2. Нажмите кнопку "QR-код" рядом с мероприятием
  3. Или перейдите по прямой ссылке на страницу QR-кода

Использование QR-кода

  1. QR-код можно распечатать или скачать для использования на мероприятии
  2. Для скачивания нажмите на меню "..." в правом нижнем углу карточки QR-кода
  3. Выберите "Скачать QR-код" или "Распечатать"
  4. Разместите QR-код в удобном месте на мероприятии
  5. Гости могут отсканировать QR-код камерой телефона для доступа к галерее

Настройка внешнего вида

Внешний вид страницы QR-кода настраивается во вкладке "Брендинг" в настройках мероприятия. Подробнее об этом читайте в разделе "Админ-панель" → "Вкладка Брендинг".

Переход к Live Фотостене

  1. На странице QR-кода нажмите на меню "..." в правом нижнем углу
  2. Выберите "Live Фотостена"
  3. Вы будете перенаправлены на страницу Live Фотостены
  4. Для возврата нажмите кнопку "Назад"

Live Фотостена

Что такое Live Фотостена

Live Фотостена - это динамическое отображение фотографий с мероприятия в режиме слайдшоу. Фотографии отображаются в случайном порядке с плавными переходами на черном фоне.

Доступ к Live Фотостене

  1. Live Фотостена доступна только для мероприятий с более чем 5 фотографиями
  2. Перейти к фотостене можно:
    • Из галереи - нажмите кнопку "Live Фотостена"
    • Со страницы QR-кода - через меню "..." → "Live Фотостена"

Как работает Live Фотостена

  1. Фотографии отображаются в случайном порядке
  2. Каждое фото отображается в течение 4 секунд
  3. Переходы между фото плавные (fade-in/fade-out)
  4. Новые фотографии появляются автоматически без перезагрузки страницы
  5. В верхней части экрана отображается название мероприятия и количество фотографий

Навигация

  1. Для возврата в галерею нажмите кнопку "Назад" (стрелка влево) в верхнем левом углу
  2. Если вы перешли на фотостену из галереи, кнопка "Назад" вернет вас в галерею
  3. Если вы перешли со страницы QR-кода, кнопка "Назад" вернет вас на страницу QR-кода

Особенности

  • Фотостена работает в полноэкранном режиме
  • Фотографии автоматически подстраиваются под размер экрана
  • Слайдшоу продолжает работать даже при сворачивании/разворачивании окна браузера
  • Обновление списка фотографий происходит автоматически каждые 5 секунд

Если у вас возникли вопросы или проблемы, обратитесь к администратору системы.
