# Инструкция по подключению Telegram уведомлений

## Шаг 1: Создание Telegram бота

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям: укажите имя и username для бота
4. После создания бота BotFather предоставит вам **токен** (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Настройка токена в проекте

1. Откройте файл `.env` в корне проекта
2. Добавьте строку с токеном:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
   ```
3. Сохраните файл и перезапустите сервер

## Шаг 3: Активация бота для получения уведомлений

1. Найдите вашего бота в Telegram (по username, который вы указали при создании)
2. Отправьте боту команду `/start`
3. Бот ответит: "✅ Уведомления настроены! Теперь вы будете получать уведомления о фото на модерации."

**Важно:** Убедитесь, что у вас установлен **username** в настройках Telegram (Настройки → Имя пользователя). Без username уведомления работать не будут.

## Шаг 4: Настройка уведомлений в админ-панели

1. Войдите в админ-панель
2. Выберите нужное событие и нажмите "Настройки"
3. Перейдите на вкладку **"Приватность"**
4. Включите чекбокс **"Включить уведомления в Telegram"**
5. Укажите ваш **Telegram username** (без символа @)
6. Установите **порог количества фото** для уведомления (например, 10)
7. Нажмите **"Сохранить"**

## Как это работает

- Когда количество фотографий на модерации превысит указанный порог, бот отправит вам уведомление в Telegram
- В сообщении будет указано название события, количество фото на модерации и ссылка на админ-панель
- Уведомления отправляются автоматически при загрузке, одобрении или отклонении фотографий

## Решение проблем

**Бот не отвечает на /start:**
- Убедитесь, что токен в `.env` указан правильно
- Проверьте, что сервер запущен и нет ошибок в консоли

**Не приходят уведомления:**
- Проверьте, что вы отправили `/start` боту
- Убедитесь, что username указан правильно (без @)
- Проверьте, что порог уведомлений настроен корректно
- Убедитесь, что уведомления включены в настройках события
